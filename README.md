# ggLowCodeGTMKit

**The complete, composable variable framework for Google Tag Manager.**

Welcome to the public repository for ggLowCodeGTMKit! This repository houses the raw source code (over 250 `.tpl` files) for the framework.

## 📚 Official Documentation

For the full interactive documentation, robust "Try Live" sandboxes, courses, and the Waitlist for the upcoming AI Agent, please visit the official site:

➡️ **[library.youdontknowga.com](https://library.youdontknowga.com)**

## 🚀 Quick Start (60 Seconds)

1. **Download:** Grab the latest bundle to access all variables.
2. **Import:** In your GTM Admin panel, go to **Import Container** and securely merge it to instantly add all 250+ variables to your workspace.
3. **Deploy:** Use the variables exactly as needed without touching any code!
